# Zeitdex

**Community resources for circadian rhythm disorders (CRDs)** — including Non-24-Hour Sleep-Wake Disorder and the delayed and advanced sleep phase disorders.

Zeitdex collects plain-language explanations, practical guidance, and a community-maintained directory of specialists and software — for and by people living with CRDs. It's the documentation companion to the **[Zeitlog](https://zeitlog.github.io/)** sleep tracker.

[:material-chart-timeline-variant: Open the Zeitlog tracker](https://zeitlog.github.io/){ .md-button .md-button--primary }
[:material-map-marker-account: Find a specialist](find-a-specialist.md){ .md-button }

!!! warning "Not medical advice"
    Zeitdex is community-maintained information, not medical advice. Always discuss your own situation with a qualified clinician.

## Contribute

Every page has an **edit** pencil that opens a pull request — no setup needed. The specialist and software directories are generated from community data in [zeitdex/resources](https://github.com/zeitdex/resources), so you can add an entry with a single JSON file. See [Contributing](contributing.md).

## Why "Zeitdex"?

From *zeitgeber* — German for "time-giver", the external cues (light, dark, routine) that set the body clock — plus *index*. The sibling tracker is **Zeitlog**.
