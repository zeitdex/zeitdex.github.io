# Zeitdex

Community resources and documentation for **circadian rhythm disorders (CRDs)**, including Non-24-Hour Sleep-Wake Disorder.

Zeitdex is the documentation companion to **[Zeitlog](https://zeitlog.github.io/)**, a browser-based sleep & circadian tracker. The site is built from Markdown with [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) and published to GitHub Pages.

## Contribute

Every page has an **Edit** (pencil) button that opens a pull request against the Markdown source — no local setup needed. See [Contributing](docs/contributing.md). Resources, corrections, and lived-experience write-ups from the CRD community are very welcome.

> Zeitdex is community-maintained information, **not medical advice**. Talk to a qualified clinician about your own situation.
